# React Examples

This package contains the React example app for the Polymarket SDK, built with Ladle.

## Scripts

From the monorepo root:

```bash
pnpm --filter @polymarket/examples-react dev
pnpm --filter @polymarket/examples-react build
pnpm --filter @polymarket/examples-react typecheck
```

## Files

- `src/` - example stories and supporting source files
- `.ladle/` - Ladle provider configuration
